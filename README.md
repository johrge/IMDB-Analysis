IMDb Movie Data Analysis

Overview
This project analyzes an IMDb movie dataset using R to identify trends in ratings, genres, popularity, and movie performance. The analysis focuses on data cleaning, exploratory data analysis (EDA), and visualization to produce clear and reproducible insights using R Markdown.

How to Run
Install required packages:
install.packages(c("tidyverse", "ggplot2", "readr", "rmarkdown"))
Open the project in RStudio.
Open IMDB Analysis.Rmd.
Click Knit to generate the report.

Output
The knitted report includes cleaned data previews, summary statistics, charts, and written insights.
